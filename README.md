# WorkHive - Shared Workspace Web App

## Project Overview

**Course:** SODV1201 - Introduction to Web Programming  
**Project Type:** Group Project (5 members)  
**Description:** A web application connecting workspace owners with coworkers, similar to Airbnb for shared workspaces.
